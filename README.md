# Nuxt-blog

个人博客，包含前台页面和后台文章管理

## 框架

- Nuxt.js
- NuxtUI
- Prisma
- Zod

## Setup

Make sure to install dependencies:

```bash
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

## Production

Build the application for production:

```bash
pnpm build
```

Locally preview production build:

```bash
pnpm preview
bun run preview
```
